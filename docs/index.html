01 — Overview

What this is

This project takes a robotic arm through the full design-to-build cycle: mechanical geometry locked in CAD, structure 3D-printed, inverse kinematics derived and coded from scratch, and a full control pipeline built layer by layer from a mouse cursor down to three physical servos. It's built as a demonstration of end-to-end product development — mechanical, electrical, and software — not just a coding exercise.

02 — Control Pipeline

Control pipeline

Five stages, each with a single responsibility. Hardware issues are fixed in firmware; math issues are fixed in the kinematics layer. Neither leaks into the other.

INPUT
Mouse + scroll
Cursor position → reach/height (cm). Scroll wheel → base angle accumulator.
kinematics.py
Solve IK
Law-of-cosines 2-link solve → shoulder/elbow servo angles. No hardware, no UI — pure geometry.
arm_serial.py
Transport
Handshakes on boot, throttles sends, ships "base,shoulder,elbow\n" @ 115200 baud.
arm_controller.ino
Translate
Pure hardware relay — parses angles, drives the PCA9685. No math lives here.
OUTPUT
3× MG995
Base, shoulder, elbow servos move to commanded angle in real time.

Full itemized BOM: Robot Arm BOM.xlsx

The Uno talks to the PCA9685 over I2C — SDA/SCL run from the analog header (A4/A5) to the driver's SDA/SCL pins, with a shared GND and 5V logic supply. Servo power is kept separate: the PCA9685's V+ rail is fed from its own DC barrel jack rather than the Uno's 5V line, so servo stall current can't brown out the microcontroller. Each MG995 plugs into its own channel (signal, V+, GND) on the driver — base, shoulder, and elbow on channels 0–2, with channel 3 reserved for the v2 claw.

04 — Engineering Decisions

Design decisions

Fix at the right layer

A physical servo inversion was corrected in firmware with a per-channel flip — not by adjusting the IK math, which would have quietly broken every other channel.

Open-loop by design

No position feedback sensors. Joint offsets are static calibration constants tuned against known physical positions — not a control-theory problem, a measurement one.

Coupon-first printing

Printing costs per gram at CAEN. Critical fit features get validated on small test coupons before committing filament to a full-link reprint.
